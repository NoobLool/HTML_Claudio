Ejercicios de autoevaluación

3.3 Complete las siguientes oraciones:

a) El atributo autofocus (que se utiliza en un solo elemento input en un formulario) resalta de manera automatica el elemento input y, si se usa de manera apropiada, coloca el cursor en el campo de texto despues de el navegador carga y despliega la pagina.

b) Los nuevos tipos de input de HTML5 tienen auto validación del lado del cliente.

c) El tipo input number al usuario introducir un valor numerico.

d) El tipo input range valida automaticamente de manera intrinseca cuando lo despliega el navegador como un control deslizante, ya que el usuario no puede mover el control fuera de los limites del valor minimo o maximo.

e) El atributo autocomplete puede usarse en los tipos input para completar de manera automatica la informacion del usuario con base en datos de entradas anteriores.

f) El elemento figcaption proporciona una leyenda para la imagen en el elemento figure.

g) El elemento summary muestra una flecha que apunta a la derecha enseguir de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, esta apunta hacia abajo y revela el contenido del elemento details.

h) El elemento mark nos permite subrayar texto.
