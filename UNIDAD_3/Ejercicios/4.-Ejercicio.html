Ejercicios de autoevaluación

3.4 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Los navegadores que despliegan el tipo input color como campo de texto requieren que el usuario introduzca el nombre de un color.
Respuesta: Falso. El usuario debe de ingresar el color como codigo hexadecimal.

b) Cuando un usuario introduce datos en un formulario y luego lo envia (por lo general, al hacer clic en el boton Enviar, el navegador verifica de inmediato que los datos sean correctos.
Respuesta: Verdadero

c) HTML5 puede validar si la direccion de correo electronico introducida por el usuario existe en realidad.
Respuesta: Falso. HTML5 no valida si una dirección email exista, si no más bien valida que este en el formato correcto.

d) Puede agregar required cualquiera de los tipos input.
Respuesta: Verdadero.

e) Puede habilitar autocomplete sólo para ciertos elementos input.
Respuesta: Falso. Se puede habilitar autocomplete para formas enteras o para elementos especificos..

f) El elemento time nos permite identificar una fecha, hora o ambas.
Respuesta: Verdadero

g) El elemento caption proporciona una leyenda para la imagen en un elemento figure.
Respuesta: Falso. El nombre del elemento es figcaption..

h) El elemento details muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer en la flecha, ésta apunta hacia abajo y revela el contenido del elemento summary.
Respuesta: Falso. El elemento summary muestra una flecha que apunta a la derecha enseguida de un resumen leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento details..

i) El elemento footer describe el contenido que por lo general aparece en la parte inferior del contenido o del elemento section
Respuesta: Verdadero.

j) El elemento highlight nos permite resaltar texto.
Respuesta: Falso. El elemento que se utiliza para subrayar es mark.
