3.2 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Cualquier tipo input particular de formulario de HTML5 debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML5.
Respuesta: Falso. El despliegue de los tipos input puede variar de un navegador a otro.

b) Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor.
Respuesta: Falso. Cuando se coloca el enfoque en el campo de texto, el texto de placeholder desaparece. No se "envía" cuando el usuario hace clic en el botón Enviat (a menos que el usuario escriba el mismo texto.

c) No es necesario incluir el atributo autofocus en sus formularios.
Respuesta: Verdadero.

d) Los nuevos tipos de input HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos invá;lidos que se enviín.
Respuesta: Verdadero.

e) El tipo input range se valida automáticamente de manera intrínseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los limites del valor minimo o maximo.
Respuesta: Verdadero.

f) HTML5 cuenta con validación automáticamente para el tipo input tel.

Respuesta: Falso. La longitud y el formato de los numeros telefonicos varia de manera considerable con base en la ubicacioon, por lo que la validacion se vuelve bastante compleja, asi que HTML5 no cuenta con validacion automatica para el tipo input tel. Para asegurar que el usuario introduzca el numero en un formato apropiado, usamos el atributo pattern

g) Si el usuario introduce un URL con un formato inapropiado en un tipo input url, no se validara. HTML5 no valida si el URL introducido existe en realidad.

Respuesta: Verdadero.

h) El elemento nav muestra un menú desplegable de hipervínculos.

Respuesta: Falso. El elemento nav agrupa vinculos de navegacion.

Respuesta: Falso. El elementoo header puede usarse varias veces en una pagina y a menudo incluye encabezados de HTML. (< h1> a < h6>).
