Ejercicios de autoevaluación

3.1 Complete las siguientes oraciones:

a) El tipo "input color" permite al usuario introducir un color. Al momento de escribir este libro, la mayoría de los navegadores despliegen el tipo "input color" como un campo de texto en donde el usuario puede introducit un código hexadecimal.

b) El atributo placeholder nos permite colocar texto temporal en un campo de texto.

c) Si desea anular la valicación, puede agregar el atributo "formvalidate" al tipo "input" submit.

d) El atributo required obliga al usuario a introducir un valor antes de enviar el formulario.

e) Por lo general se muestra el control spinner para el "input number" e incluye sólo los números válidos.

f) El tipo "input" time permite el usuario introducir una hora, minuto, segundo y fracción de segundo.

g) El elemento datalist proporciona opciones de entrada para un elemento "input text".

h) El elemento article describe contenido que está separado del contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artílo de noticias, mensaje de foro o entrada de blog..

i) El elemento footer describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento "section".

j) El elemento wbr indica el lugar apropiado para dividir una palabra cuando el texto se ajusta en varias líneas.
