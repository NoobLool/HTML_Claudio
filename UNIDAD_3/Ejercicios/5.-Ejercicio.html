3.5 Escriba un elemento (o elementos) de HTML5 para realizar cada una de las siguientes tareas:

a) Se pidió a los estudiantes que clasificaran los alimentos en la cafeteria, en una escala del 1 al 10. Use un elemento meter con texto a su izquierda y a su derecha para indicar que la calificación fue de 7 de 10.

Respuesta: < p >1 < meter min = "1" max = "10" value = "7" >< /meter> 10 < /p>< /meter>

b) Cree un elemento details que muestre el texto "Resultados de la encuesta" en el elemento summary para la parte (a). Cuando el usuario haga clic en la flecha enseguida del texto summary, deberá aparecer un párrafo que explique sobre la encuesta.

Respuesta: < details > < summary > Resultados de la encuesta < /summary> < p > La calificación que se le dio a la cafeteria en la escala del 1 al 10. Fue 7. < /p > < /details >

c) Cree un elemento input text para el primer nombre. El elemento deberá recibir el enfoque de manera automática al momento de desplegar el formulario en un navegadoor.

Respuesta: < p > < label > Nombre: < input type = "text" autofocus /> < /label > < /p >

d) Modifique la parte (c) de modo que se elimine el elemento label y use texto en un elemento placeholder en el elemento input

Respuesta: < p > < input type = "text" placeholder = "Nombre" autofocus /> < /p >
